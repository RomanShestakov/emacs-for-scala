emacs-for-scala
===============

My Emacs setup for Scala development.

The .emacs.d contained in this repo has a specific set of keybindings set up for Emacs. Mostly these are the defaults from whatever included modules have been used, but in a few cases they've been customized to play well together. I'm trying to make this section a comprehsive guide to the major key bindings you'll need when working with Emacs and developing Scala code.

Please see <a href="http://jglobal.com/emacs-for-scala/" target="_new">http://jglobal.com/emacs-for-scala/</a> for a series of articles describing the Emacs config I've published here.
